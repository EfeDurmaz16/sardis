# Getting Started

This guide shows how to set up credentials, run the API locally, and make your first payment request.

## Prerequisites
- Python 3.10+
- `uvicorn` and Sardis packages installed (editable install)
- `.env` configured (see `TESTING_GUIDE.md` for full env vars)

## Install
```bash
# from repo root
pip install -e packages/sardis-core -e packages/sardis-protocol -e packages/sardis-wallet \
  -e packages/sardis-chain -e packages/sardis-ledger -e packages/sardis-compliance \
  -e packages/sardis-cards -e packages/sardis-api
```

## Run API locally
```bash
uvicorn sardis_api.main:create_app --factory --host 0.0.0.0 --port 8000
```
Docs: http://localhost:8000/api/v2/docs

## First call: health
```bash
curl http://localhost:8000/
```
