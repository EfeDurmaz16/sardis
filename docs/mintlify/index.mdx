# Sardis API Docs

Sardis is the payment execution layer for AP2/TAP agents, enabling stablecoin and virtual card payments with mandate enforcement, MPC custody, and compliance.

## What you can do
- Execute AP2 payment mandates across Base, Polygon, Ethereum
- Issue and fund virtual cards (Lithic) for fiat rails
- Run compliance (KYC, sanctions) checks inline
- Monitor transactions and audit logs
