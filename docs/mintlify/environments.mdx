# Environments & Keys

Sardis supports simulated (dev) and live modes. Configure via `.env`.

## Core variables
- `SARDIS_ENVIRONMENT`: `dev` | `sandbox` | `prod`
- `SARDIS_CHAIN_MODE`: `simulated` (no real tx) or `live`
- `DATABASE_URL`: SQLite for dev, PostgreSQL for prod
- `SARDIS_ALLOWED_ORIGINS`: CORS origins

## MPC (Turnkey)
- `TURNKEY_ORGANIZATION_ID`
- `TURNKEY_API_PUBLIC_KEY`
- `TURNKEY_API_PRIVATE_KEY`

## Compliance
- `PERSONA_API_KEY`, `PERSONA_TEMPLATE_ID`
- `ELLIPTIC_API_KEY`, `ELLIPTIC_API_SECRET`

## Cards (Lithic)
- `LITHIC_API_KEY`
- `LITHIC_ENVIRONMENT`: `sandbox` or `production`

## Chain RPC
- `BASE_SEPOLIA_RPC_URL`, `POLYGON_AMOY_RPC_URL`, `SEPOLIA_RPC_URL`, etc.

For the full list, see `TESTING_GUIDE.md` in the repo.
