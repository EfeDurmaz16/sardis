# Disclosures and Agreements Checklist

Date: 2026-03-04

## Hosted Onboarding First

Default onboarding strategy:

1. Prefer Stripe-hosted onboarding before custom embedded onboarding.
2. Keep custom KYC collection disabled until compliance/legal signoff.
3. Track substantive copy changes through issuer/compliance review flow.

## Required Issuing Baseline

Before enabling live issuance, confirm acceptance/presentation flow for:

1. Platform terms and connected account agreement.
2. Issuing accountholder terms.
3. Issuing bank terms for the selected sponsor bank.
4. E-sign consent language near submission action.
5. Commercial financing disclosures where jurisdictionally required.
6. Authorized user terms (if individual cardholders are used).

## Change Management

1. Store approved disclosure copy and version tag in this repo.
2. Route pricing/fee copy changes through compliance review.
3. Submit formal change requests where issuer/program requires it.
4. Keep an audit trail with timestamp, approver, and affected surfaces.
