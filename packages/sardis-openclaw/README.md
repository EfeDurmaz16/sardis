# sardis-openclaw

OpenClaw skill package for Sardis - Payment OS for AI Agents.

## What is this?

This package provides the OpenClaw skill definition for Sardis, enabling AI agents
in the OpenClaw ecosystem to manage wallets, execute payments, and enforce spending policies.

## Installation

```bash
pip install sardis
```

## Usage

The SKILL.md file in this package defines the Sardis skill for OpenClaw. It includes:
- CLI command documentation
- Python SDK usage examples
- Framework integration guides
- Supported chains and tokens

## Publishing to ClawHub

To publish this skill to the OpenClaw ClawHub:

1. Verify your OpenClaw account
2. Navigate to ClawHub submission page
3. Upload the SKILL.md file
4. Set category: payments, fintech, infrastructure
5. Submit for review
