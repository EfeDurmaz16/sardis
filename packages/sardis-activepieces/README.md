# @activepieces/piece-sardis

Sardis payment piece for [Activepieces](https://activepieces.com) — policy-controlled payments for AI agents.

## Install

```bash
npm install @activepieces/piece-sardis
```

## Usage

1. Add the Sardis piece to your Activepieces instance
2. Configure your API key in the connection settings
3. Use the **Send Payment** or **Check Balance** actions in your flows

## Actions

| Action | Description |
|--------|-------------|
| Send Payment | Execute a policy-controlled stablecoin payment |
| Check Balance | Check wallet balance and spending limits |

## Authentication

Requires a Sardis API key (`sk_...`). Get one at [sardis.sh](https://sardis.sh).
