# create-sardis-app

Scaffold a new Next.js + Vercel AI SDK + Sardis application.

## Usage

```bash
npx create-sardis-app my-agent-app
cd my-agent-app
npm install
npm run dev
```

## What's included

- Next.js 16 app router scaffold
- Vercel AI SDK with Sardis provider (`sardis/ai-sdk`)
- Pre-wired API key flow
- Working example: `/api/chat` route with `sardis.pay()` tool
