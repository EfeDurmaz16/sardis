> [Sardis](https://sardis.sh) is the Payment OS for the Agent Economy — infrastructure
> enabling AI agents to make real financial transactions safely through non-custodial
> MPC wallets with natural language spending policies.

With Sardis, your LangChain agents can:

- Send stablecoin payments (USDC, USDT, EURC) across 6+ EVM chains
- Enforce spending policies before every transaction
- Operate non-custodially via MPC wallets (no private keys stored)
- Produce an append-only audit trail for every payment

Supported chains: Arc (Circle L1), Base, Polygon, Ethereum, Arbitrum, Optimism.

## Installation and setup

<CodeGroup>
```bash pip
pip install sardis-langchain
```

```bash uv
uv add sardis-langchain
```
</CodeGroup>

Get a Sardis API key at [sardis.sh](https://sardis.sh) and export it:

```bash
export SARDIS_API_KEY="sk_..."
export SARDIS_WALLET_ID="wal_..."
```

## Tools

```python
from sardis_langchain import SardisPaymentTool, SardisBalanceTool, SardisPolicyCheckTool
```

### SardisPaymentTool

Execute policy-controlled payments from an AI agent's wallet.

```python
from sardis_langchain import SardisPaymentTool

tool = SardisPaymentTool()
result = tool.invoke({
    "amount": 25.00,
    "merchant": "openai",
    "purpose": "API credits"
})
```

### SardisBalanceTool

Check wallet balance and remaining spending limits.

```python
from sardis_langchain import SardisBalanceTool

tool = SardisBalanceTool()
result = tool.invoke({"token": "USDC"})
```

### SardisPolicyCheckTool

Pre-check if a payment would be allowed by spending policy.

```python
from sardis_langchain import SardisPolicyCheckTool

tool = SardisPolicyCheckTool()
result = tool.invoke({
    "amount": 500.00,
    "merchant": "aws"
})
```

## Use within an agent

```python
from langchain.chat_models import init_chat_model
from sardis_langchain import SardisPaymentTool, SardisBalanceTool

tools = [SardisPaymentTool(), SardisBalanceTool()]
model = init_chat_model("gpt-4o", model_provider="openai")

from langgraph.prebuilt import create_react_agent
agent = create_react_agent(model, tools)

for step in agent.stream(
    {"messages": [("human", "Check my balance and pay $50 to Vercel for hosting")]},
    stream_mode="values",
):
    step["messages"][-1].pretty_print()
```
